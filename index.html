The pipeline

photo + shorthand→ Qwen3.5-9B expander (structure + guardrails)→ Qwen-Image-Edit-2511 (NF4, Lightning 8-step)→ procedure LoRA (QLoRA on real pairs)→ eval canary (DINO · ArcFace · LPIPS · CLIP)

The whole system — inference, LoRA training, and evaluation — runs on one 32 GB consumer GPU. The original hackathon build died from silent identity collapse (the model learned to return its input); the rebuild's edit-magnitude canary makes that failure impossible to miss, and you can see it reporting under every result above.

For consultation-research purposes only — not a medical device. source: github.com/munishbp/AURA
